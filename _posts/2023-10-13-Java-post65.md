# 포스코DX X 비트교육센터 6기 - FRONT

---

## 종속 선택자 

- 태그, 클래스, 아이디 선택자가 결합되 형태의 선택자

- 종속 선택자 예

```
  h1#head { … } 
  .headline.selected { … }
  input#user-id.focused { …}
  p.title { … }
```

## 하위 선택자

- 선택자 내부의 자식 선택자에 속성을 지정하는 방식이다.
- 하위 선택자 예

```
  body h1, body h2, body p { … } 
  p .txt1 { … }                                                
  .headline span                                            
```
